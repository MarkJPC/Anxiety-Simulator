## Build Commands:


#### To make everything run:
```make
make all
```
> **Note:**
> does not run clean or doxygen
<br/>

#### To make tests run:
```make
make test
```
<br/>

#### To make build run:
```make
make build
```
<br/>

#### To clean the directory run:
```make
make clean
```
